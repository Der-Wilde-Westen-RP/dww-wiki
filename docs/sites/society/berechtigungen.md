# BERECHTIGUNGEN

Bei einem Gewerbe (sowie jedem anderen Job, dort allerdings abweichend) gibt es verschiedene Berechtigungen, die vom Inhaber sowie dafür freigeschalteten Personen eingestellt werden können.
Dabei kann für jede Person die Berechtigung individuell angepasst werden. Dies erfolgt über das F6-Menü -> Zugänge -> Zugänge.

Die Berechtigung für die Auslage kann in jedem Betrieb selbst eingestellt werden.
Ein Anpassen der Berechtigungen ist mit ausreichender Begründung im Support auf unserem [Discord](https://discord.gg/dww-rp) möglich.
